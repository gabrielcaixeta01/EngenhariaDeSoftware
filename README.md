# Engenharia de Software - UnB

Este repositório contém os materiais, códigos e práticas desenvolvidos ao longo da disciplina **Engenharia de Software** da Universidade de Brasília (UnB).

## 📌 Objetivo da disciplina
A matéria busca ensinar **como construir software de forma eficaz**, aplicando conceitos fundamentais como:
- Desenvolvimento utilizando **Ruby on Rails**;
- Criação de um sistema **modularizado e escalável**;
- **Testes automatizados** para garantir qualidade;
- Boas práticas de versionamento e documentação;
- Conceitos de arquitetura e padrões de projeto.

## 🛠️ Tecnologias utilizadas
- **Ruby**  
- **Ruby on Rails**  
- **Git/GitHub** (versionamento)  
- **Rspec/Minitest** (testes automatizados)  

## 📂 Estrutura do repositório
O repositório será organizado conforme as atividades da disciplina, incluindo:
- Implementações de projetos em Rails
- Exercícios práticos
- Testes automatizados
- Documentação complementar

## 🎯 O que vamos estudar
Ao longo do semestre, serão abordados temas como:
- Ciclo de vida do software
- Engenharia de requisitos
- Arquitetura de software
- Testes e qualidade de software
- Metodologias ágeis
- Modularização e boas práticas de desenvolvimento

---

📖 Este repositório servirá como base de estudos e prática para consolidar os conceitos da disciplina e construir software de maneira **eficiente, organizada e com qualidade**.
